Squishy Sidebar (Getting Pushy)

This example demonstrates sidebars that squish the entire page (header, body and footer)

Features:
  • Panels animate in (CSS ANIMATION)
  • Panels "pop in" if browser does not support CSS animations
  • All page sections controlled now with FULL bootstrap columns
  • Body resizes Reactively
  • Footer resizes reactively
  • JS required for functionality
Caveat(s)
  • Navbar CANNOT use bootstrap fixed positioning
  • Dropped CSS only support to provide cleaner html solution
  • JS is now required
  • Example is jQuery, but can be easily converted to your framework or pure JS
Bug(s)
  1. Panels expressed with left and right using absolute positioning. This causes the right panel to increase the page width, offscreen. Users who use mouse scrolling will be able to see this
